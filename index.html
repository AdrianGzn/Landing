Inicio

Somos una mricroempresa llamada A&S.

Esta microempresa está conformada por Sergio Adrián Guzmán Pérez y Héctor Emilio Somer Velázquez.

Somos ingenieros de la Universidad Politécnica de Chiapas cursando la ingeniería en Desarrollo de Software bajo la enseñansa del profesor Luis Gutiérrez Alfaro.

Ofrecemos servicios de Software tales orientados a:

  • C++

  • HTML

  • Python

  • MySQL

  • php

  • JavaScript

Le ofrecemos estos servicios a un precio muy accesible, los cuales los puede consultar en la sección de "Cotizar precios".





Servicios

  • C++:

    • En el lenguaje C++ nos especializamos en resolver problemas de lógica algorítmica haciéndo uso de consola aplicada.

  • HTML:

    • Trabajamos en la creación de páginas web con Lenguaje de Etiquetas de Hipertexto, enfocándonos en tener una página estética a gusto personal (esto se espacifica en el apartado Cotizar Precios).

  • Python:

    • Dentro del lenguaje Python trabajamos en desarrollo de videojuegos de tu preferencia, ya sea personal o comercial.

  • MySQL:

    • Añadiendo MySQL a HTML, podemos crear páginas con ayudad de bases de datos para gestionar usuarios o inventarios.

  • php y JavaScript:

    • Volvemos laspáginas más dinámicas, más estilizadas y que tengan más interacción con el usuario.

    Se requiere con anticipación una propuesta de trabajo para un precio más accesible.





Cotizar Precios

  • C++:

    • El precio está directamente relacionado con la dificultad del algoritmo a resolver. El precio ronda entre los $100 hasta los $1,000.

  • HTML:

    • El precio del dominio de la página corre por cuenta del usuario. Debido al proceso más largo que conlleva esto (diseñarla y ponerla en línea), el precio ronda entre $500 y $2,000 (Puede se más elevado dependiendo de las especificaciones).

  • Python:

    • Precio a tratar, aunque considerablemente más elevado que los demás (precio para aplicaciones comerciales más elevado).

  • MySQL:

    • El precio dependerá de la capacidad de almacenamiento de la base de datos proporcionada.

  • php:

    • El precio es proporcional a la dificultad del trabajo y la anticipación con la que se haya pedido.

  • JavaScript

    • En JavaScript dependerá del tipo del diseño que se le pida a la página, ya que se trabajará con una página previamente hecha, a la cual se le dará el diseño y animaciones.

    Ronda entre los $500 y $1,000.
